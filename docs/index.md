## The suite

This repo is one of several siblings. Each defends its own falsifier and ships under its own version.

- [flywheel-concept](https://github.com/velvetmonkey/flywheel-concept) (this repo) — research programme on whether AI representation reveals shared concept geometry
- [flywheel-geometry](https://github.com/velvetmonkey/flywheel-geometry) — instrument-calibration / Method 6 falsifier
- [flywheel-memory](https://github.com/velvetmonkey/flywheel-memory) — graph-native memory layer for AI systems
- [flywheel-crank](https://github.com/velvetmonkey/flywheel-crank) — Obsidian plugin for vault navigation
- [vault-core](https://github.com/velvetmonkey/vault-core) — supporting infrastructure

The repos are siblings, not nested. Concept does not own Geometry. Memory is its own product. Each falsifier stands or falls on its own evidence.
